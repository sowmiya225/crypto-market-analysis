# Cryptocurrency Market Analysis using API

## Project Overview
This project analyzes real-time cryptocurrency market data using the CoinGecko API. The goal is to extract cryptocurrency information, analyze market capitalization and trading volume, and visualize insights.

## Technologies Used
- Python
- Pandas
- Matplotlib
- API Integration

## Data Source
CoinGecko API

## Analysis Performed
- Top 10 cryptocurrencies by Market Capitalization
- Top 10 cryptocurrencies by Trading Volume
- Relationship between Price and Market Cap

## Project Files
crypto_analysis.py – Python script for API data extraction and analysis  
crypto_data.csv – Dataset generated after running the script  
requirements.txt – Python libraries required to run the project  

## To Run the Project

Install required libraries

pip install -r requirements.txt

Run the Python script

python crypto_analysis.py
